# YouMeal-CleanPlan
北京邮电大学雏雁计划 - "邮"你清盘# "邮"你清盘 - 后端API服务

北京邮电大学2025年雏雁计划项目

## 项目简介
智能餐盘剩菜统计系统，通过AI图像识别技术分析食堂浪费情况。

## 技术栈
- Python Flask
- SQLite数据库
- JWT用户认证
- RESTful API设计

## 快速开始

### 1. 环境配置
```bash
# 克隆项目
git clone https://github.com/你的用户名/youmeal-backend.git
cd youmeal-backend

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
